# 什么是响应式设计？响应式设计的基本原理是什么？如何兼容低版本的IE？

响应式设计（Responsive Web design）：集中创建页面的图片排版大小，可以智能的根据用户使用的设备环境（系统平台、屏幕尺寸、屏幕定向等）进行对应的布局。
基本原理：使用@media媒体查询，然后使用不同的CSS规则来实现屏幕的最佳布局。
